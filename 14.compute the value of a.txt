14. Write a program that computes the value of a+aa+aaa+aaaa with a given digit as the value of a.
Suppose the following input is supplied to the program:
9
Then, the output should be:
11106

code:
a = input()
v1 = int( "%s" % a )
v2 = int( "%s%s" % (a,a) )
v3 = int( "%s%s%s" % (a,a,a) )
v4 = int( "%s%s%s%s" % (a,a,a,a) )
print(v1+v2+v3+v4)

input:
9

output:
11106
